# TilesGallery
A modern tile-based web application built with Next.js, designed to showcase dynamic UI components, authentication flow, and responsive design.

## Live Demo
https://tiles-gallery-brown-psi.vercel.app

## Project Purpose
  TileScape is a frontend-focused project built to practice and demonstrate:
  
  - Modern Next.js App Router architecture
  - Responsive UI design using Tailwind CSS
  - Authentication handling (login/logout/profile flow)
  - Dynamic component rendering (tiles system)
  - Clean folder structure for scalable React apps

## Key Features
- Responsive tile-based UI layout
- Authentication system (login/logout/profile handling)
- Fast performance with Next.js App Router
- Modern UI with Tailwind CSS
- Search/filter functionality for tiles
- Mobile-friendly design
- Custom 404 page with animated background
- Deployed on Vercel

## NPM Packages Used
### Core Framework
- next – React framework for production
- react – UI library
- react-dom – React DOM rendering
### UI & Styling
- tailwindcss – Utility-first CSS framework
- postcss – CSS processing
- autoprefixer – CSS compatibility tool
### Icons & UI Enhancements
